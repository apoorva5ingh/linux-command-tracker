# Linux Command Usage Tracker

A simple Bash-based utility that tracks and analyzes the commands you use in the Linux terminal.

This project reads your shell history and shows which commands you use the most, helping you understand your command-line habits.

---

## Features

- Tracks command usage from shell history
- Displays top most-used commands
- Supports daily usage logs
- Works with both **Bash** and **Zsh**
- Lightweight and terminal-based
